### 项目简介

TDesign Vue Next Starter 是一个基于 TDesign，使用 `Vue3`、`Vite3`、`TypeScript` 开发，可进行个性化主题配置，旨在提供项目开箱即用的、配置式的初始项目。

<p>
  ·
  <a href="https://tdesign.tencent.com/starter/">使用文档</a>

</p>

### 开发

```bash
## 安装依赖
npm install

## 启动项目
npm run dev
```

```bash
