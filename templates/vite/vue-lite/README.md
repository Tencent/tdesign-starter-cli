## 项目简介

`tdesign-vue` 是一个 TDesign 适配桌面端的组件库，适合在 vue 2 技术栈项目中使用。

## 开发

### 安装依赖

```bash
npm install
```

### 启动项目

```bash
npm run dev
```

## 构建

### 构建正式环境

```bash
npm run build
```
