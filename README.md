## Install

```shell
npm install tdesign-starter-cli -g
```

## Usage

```sh
# project initialization
td-starter init

# fill project name and description
? 请输入项目名称： [project-name]
? 请输入项目描述： Base on tdesign-starter-cli

cd [project-name]

# install dependencies
npm install

# Mac OS or Windows
npm run dev 

```
