## 安装

```shell
npm install tdesign-starter-cli -g
```

## 使用

```sh
# 初始化项目
td-starter init

# 填写 项目名称、项目描述
? 请输入项目名称： [项目名称]
? 请输入项目描述： Base on tdesign-starter-cli

# 进入项目
cd [项目名称]

# 安装依赖
npm install

# 运行
npm run dev 

```
